# BrandBrain - 品牌销量预测系统

基于 BrandFormer 深度学习模型的品牌销量预测与决策模拟系统。

## 功能模块

1. **数据导入** - 支持 CSV/Excel 上传
2. **数据预处理** - 缺失值处理、异常检测、特征工程
3. **模型训练** - BrandFormer + LightGBM 混合模型
4. **预测分析** - 销量预测 + SHAP 解释 + 注意力热图
5. **决策模拟** - 场景模拟与策略推荐
6. **智能对话** - Claude AI 助手
